# 🐾 PetVerse AI

PetVerse AI is a futuristic AI-powered pet care ecosystem designed to help pet owners manage everything in one intelligent platform.

It combines:
- 🤖 AI Pet Assistant
- 🐶 Interactive 3D Virtual Pets
- 🏥 Emergency Vet Support
- 🛒 Pet Marketplace
- 📍 Nearby Vet Finder
- 📊 Smart Health Tracking
- 🧠 AI Health Intelligence

Built with modern technologies like Next.js, React Three Fiber, OpenAI, Firebase, and Google Maps API.

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/petverse-ai.git
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Run Development Server

```bash
npm run dev
```

Open:
```bash
http://localhost:3000
```

---

# 🔑 Environment Variables

Create `.env.local`

```env
OPENAI_API_KEY=
NEXT_PUBLIC_FIREBASE_API_KEY=
MONGODB_URI=
GOOGLE_MAPS_API_KEY=
STRIPE_SECRET_KEY=
```
